Animaciones CSS y Bootstrap

Guía completa de transiciones, animaciones y efectos

1. TRANSICIONES CSS (transition)

Las transiciones suavizan los cambios entre estados CSS (hover, focus, etc.)

Botones con transiciones

/* SINTAXIS */ transition: property duration timing-function delay; /* EJEMPLO */ .btn-transition { transition: all 0.3s ease; } .btn-transition:hover { background-color: blue; transform: translateY(-5px); } /* PROPIEDADES INDIVIDUALES */ .btn { transition-property: background-color, transform; transition-duration: 0.3s; transition-timing-function: ease; transition-delay: 0s; }

Cards con transición

Hover Básico

Hover Elevación

Se eleva con sombra pronunciada.

Hover Escala

Crece ligeramente al pasar el mouse.

2. ANIMACIONES @keyframes

Las animaciones @keyframes permiten crear movimientos complejos y repetitivos.

Ejemplos de animaciones

Fade In

Aparece gradualmente

Slide In

Entra desde abajo

Bounce

Rebota continuamente

Pulse

Pulsa continuamente

/* SINTAXIS DE @keyframes */ @keyframes nombre-animacion { from { /* Estado inicial */ } to { /* Estado final */ } } /* O con porcentajes para más control */ @keyframes fade-in { 0% { opacity: 0; transform: translateY(20px); } 100% { opacity: 1; transform: translateY(0); } } /* APLICAR ANIMACIÓN */ .elemento { animation: fade-in 1s ease-out; } /* PROPIEDADES DE ANIMATION */ .elemento { animation-name: fade-in; animation-duration: 1s; animation-timing-function: ease-out; animation-delay: 0s; animation-iteration-count: 1; /* O infinite */ animation-direction: normal; animation-fill-mode: forwards; }

3. TIMING FUNCTIONS (Funciones de tiempo)

Controlan la velocidad de la animación a lo largo del tiempo.

Compara las diferentes funciones (Haz clic en "Animar")

linear
ease (defecto)
ease-in
ease-out
ease-in-out
/* TIMING FUNCTIONS */ linear - Velocidad constante ease - Lento al inicio y final (defecto) ease-in - Lento al inicio, rápido al final ease-out - Rápido al inicio, lento al final ease-in-out - Lento al inicio y final /* CUSTOM (cubic-bezier) */ transition: all 1s cubic-bezier(0.68, -0.55, 0.265, 1.55);

4. TRANSFORM (Transformaciones)

Permite rotar, escalar, mover y sesgar elementos sin afectar el flujo del documento.

Transformaciones básicas

translate

translateY(-10px)

rotate

rotate(15deg)

scale

scale(1.2)

skew

skew(10deg)
/* TRANSFORM - TRANSLATE (Mover) */ transform: translateX(50px); /* Mover horizontal */ transform: translateY(-20px); /* Mover vertical */ transform: translate(50px, -20px); /* Ambos */ /* TRANSFORM - SCALE (Escalar) */ transform: scale(1.5); /* Agrandar 150% */ transform: scale(0.5); /* Reducir 50% */ transform: scale(1.2, 0.8); /* X e Y diferentes */ /* TRANSFORM - ROTATE (Rotar) */ transform: rotate(45deg); /* Rotar 45 grados */ transform: rotate(-90deg); /* Rotar -90 grados */ /* TRANSFORM - SKEW (Sesgar) */ transform: skewX(10deg); /* Sesgar horizontal */ transform: skewY(5deg); /* Sesgar vertical */ /* COMBINAR TRANSFORMACIONES */ transform: translate(50px, -20px) rotate(45deg) scale(1.2);

TIPS IMPORTANTES PARA EL EXAMEN

  • Transitions: Usa all 0.3s ease para efectos hover suaves
  • Transforms: No afecta el flujo del documento (mejor que margin/padding para animaciones)
  • @keyframes: Para animaciones complejas o infinitas
  • Performance: Anima solo transform y opacity para mejor rendimiento
  • Timing: ease-out para hover, ease-in-out para animaciones
  • Bootstrap: Usa spinner-border para indicadores de carga
