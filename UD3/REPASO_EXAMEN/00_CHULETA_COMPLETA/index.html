Clase Significado Tamaño

mt-3 Margin top 1rem

mx-auto Margin horizontal auto (centrar) auto
mt-auto Margin top auto (empujar abajo en flexbox) auto

py-3 Padding vertical 1rem

Tamaños: 0=0, 1=0.25rem, 2=0.5rem, 3=1rem, 4=1.5rem, 5=3rem

Display y Responsive

<!-- Ocultar en móvil, mostrar en tablet+ --> <div class="d-none d-md-block">Visible en tablet+</div> <!-- Mostrar en móvil, ocultar en tablet+ --> <div class="d-block d-md-none">Solo móvil</div> <!-- Flexbox solo en desktop --> <div class="d-none d-lg-flex">Flex en desktop</div>

Clase Efecto

4. SASS BÁSICO

Variables

$color-primary: #667eea; $spacing: 1rem; .button { background-color: $color-primary; padding: $spacing; }

Nesting (Anidamiento)

.card { background: white; padding: 20px; &__title { font-size: 24px; } &:hover { box-shadow: 0 4px 8px rgba(0,0,0,0.2); } }

Mixins

@mixin button-style($bg-color) { background-color: $bg-color; padding: 10px 20px; border-radius: 5px; } .btn-primary { @include button-style(#007bff); }

Personalizar Bootstrap

// 1. Variables ANTES de importar $primary: #667eea; $secondary: #764ba2; // 2. Importar Bootstrap @import "~bootstrap/scss/bootstrap"; // 3. Tus estilos personalizados

5. ANIMACIONES Y TRANSICIONES

Transition (para hover)

.btn { transition: all 0.3s ease; } .btn:hover { transform: translateY(-5px); box-shadow: 0 10px 20px rgba(0,0,0,0.3); }

Animation @keyframes

@keyframes fadeIn { from { opacity: 0; transform: translateY(20px); } to { opacity: 1; transform: translateY(0); } } .fade-in { animation: fadeIn 1s ease-out; }

Transform (las 4 básicas)

transform: translate(50px, -20px); /* Mover */ transform: scale(1.2); /* Escalar */ transform: rotate(45deg); /* Rotar */ transform: skew(10deg); /* Sesgar */ /* Combinar */ transform: translate(50px) rotate(45deg) scale(1.2);

Timing Functions

ease /* Defecto, suave inicio y final */ ease-in /* Lento inicio */ ease-out /* Lento final (para hover) */ ease-in-out /* Lento inicio y final */ linear /* Velocidad constante */

6. TIPS PARA EL EXAMEN

Estructura HTML básica

<!DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Título</title> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> </head> <body> <!-- Contenido --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script> </body> </html>

Errores comunes a evitar

  • Olvidar container o row en el grid
  • No usar h-100 para cards con altura igual
  • Olvidar data-bs-toggle y data-bs-target en modales
  • No inicializar tooltips con JavaScript
  • Usar col- sin especificar el número (1-12)
  • Olvidar navbar-toggler en navbar responsive

Combinaciones más usadas

  • Card con altura igual: h-100 d-flex flex-column + mt-auto en botón
  • Centrado perfecto: d-flex justify-content-center align-items-center
  • Ocultar en móvil: d-none d-md-block
  • Espaciado vertical: mb-3 entre elementos
  • Transición suave: transition: all 0.3s ease;

Checklist del examen

  • ¿Incluí Bootstrap CDN en el <head>?
  • ¿Incluí Bootstrap JS antes de </body>?
  • ¿Usé container y row correctamente?
  • ¿Las cards tienen h-100 y mt-auto?
  • ¿La navbar es responsive con navbar-toggler?
  • ¿Los modales tienen IDs únicos?
  • ¿El formulario tiene validación con needs-validation?
  • ¿Las animaciones son suaves y tienen transition?
